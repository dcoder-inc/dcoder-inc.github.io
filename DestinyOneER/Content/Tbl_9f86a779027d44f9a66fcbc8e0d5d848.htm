presentation_props 
Definition  
This table represents presentation properties that can be used to customize the different views within OneCE, by configuring different entities available (ex. Term, Program Area colors)
 

presentation_props Attributes 
Attribute/Logical Rolename  Domain  Datatype  NULL  Definition 
presentation_props_id    BIGINT  NO  
ID of the presentation property (system generated)
 
web_apps_type    VARCHAR(50)  NO  
Different views where the presentation element will be available: Public Site, Internal Site, Bridge View
 
entity_class_name    VARCHAR(200)  NO  
The java class name of the entity (ex. Semester).  We will be customizing the color of the term/semester. Refers to the part of the entity that will be customized.
 
entity_object_id    BIGINT  NO  
ID of the entity (system generated ID of the term, for example)
 
entity_attribue_name    VARCHAR(50)  NO  
The attribute name of the entity we are customizing - to match the java attribute name
 
version    BIGINT  NO  
The number of times the record has been updated
 
css_style    VARCHAR(512)  YES  
Used for more generic style changes - you can include the CSS script to further customize the entity
 
background_color    VARCHAR(30)  YES  
The background color for the property - not necessary if using css_style
