public_view_selection_mode 
Base Entity  section_special_request 
ColumnName  public_view_selection_mode 
Definition  
Selection mode of the special request when it's applicable on public view enrollment page. 'Mandatory' means the special request is a mandatory fee in enrollment, 'Optional' means the special request is an optional choice in enrollment.
 
b_default_selected 
Base Entity  section_special_request 
ColumnName  b_default_selected 
Definition  
A boolean meaning flag to indicate if the special request should be selected by default if 'public_view_selection_mode' is 'Optional'
