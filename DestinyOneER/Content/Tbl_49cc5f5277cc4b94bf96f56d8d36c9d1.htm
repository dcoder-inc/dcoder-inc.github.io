hold_reason_stdnt_prtl_rpt 
Definition  
This table contains the restricted reports for student portal per hold reason.  Each report per hold reason listed here will need to be checked when a user attempts to generate the report for a student with a hold reason.  There will be one record per hold reason and report combination.
