tuition_special_request

Special Requests associated with a Tuition Profile

position    INTEGER  NO  
Order of Special Request
 
application_rule    VARCHAR(100)  YES  
Rule on how to apply the special request. 'always' means the Special Request will always be applied when the associated Tuition Profile is selected in enrollment; 'OnFirstEverEnrollment' means the Special Request is available only when the Special Request has never been purchased before by that student; 'OnFirstEnrollmentEachTerm' means the Special Request has never been purchased in current term by that student
 
student_type    VARCHAR(100)  YES  
Type of Location of the student. Empty string means ignore location type, 'Remote', 'Local' and 'Verify' which means only if a student's location matches to the value here,  the special request will be applicable for her/him.
