credit_type 
Base Entity  section_fee_tuition 
ColumnName  credit_type 
Definition  
Credit Type that will be applied when this tuition is choosed in enrollment of associated course section. 'creditInterval' means a user can pick a unit from minimum units to maximum units in a specified interval. 'nonCredit' means a user will not get a unit option when she/he is enrolling the course section with assoicated tuition.
