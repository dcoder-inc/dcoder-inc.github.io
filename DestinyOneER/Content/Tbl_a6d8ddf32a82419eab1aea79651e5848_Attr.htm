enrollment_restriction_id 
Base Entity  enrollment_restriction 
ColumnName  enrollment_restriction_id 
Definition  
The ID of the enrollment restriction record
 
parent_id 
Base Entity  enrollment_restriction 
ColumnName  parent_id 
Definition  
Either a course ID or a course section profile ID which indicates to which course or course section profile  this enrollment restriction applies
 
applicant_type 
Base Entity  enrollment_restriction 
ColumnName  applicant_type 
Definition  
The type of applicant to which this enrollment restriction should apply - either "Student" or "Group"
 
restriction_type 
Base Entity  enrollment_restriction 
ColumnName  restriction_type 
Definition  
The type of the enrollment restriction - "regular" means an enrollment restriction will be applied to all types of fees when a course section profile is enrolled, "discoun" means the restriction will only be applied on those discounts applicable when a course section profile is enrolled, "'surcharge" means the restriction will only be applied on those surcharges applicable when a course section profile is enrolled, "tuition profile" means the restriction will only be applied on those tuition profiles applicable when a course section profile is enrolled
 
restriction_object_id 
Base Entity  enrollment_restriction 
ColumnName  restriction_object_id 
Definition  
The ID of the fee items (tuition profile, discount, surcharge) to which a enrollment restriction will apply - 1 means "all" - other valid ID values should be a foreign key to the FEE table
 
application_rule 
Base Entity  enrollment_restriction 
ColumnName  application_rule 
Definition  
The rule of association for the application profile - "mustSubmittedApplication" means this enrollment restriction will force a user to submit an appliation for the associated course section profile that is to be enrolled; "mustApprovedApplication" means this enrollment restriction will force a user to have an approved application for the associated course section profile that is to be enrolled; "autoAddApplication" means an associated application will be added automatically after the enrollment transaction basket is processed
 
application_profile_id 
Base Entity  enrollment_restriction 
ColumnName  application_profile_id 
Definition  
The ID of the associated application profile - a foreign key to the APPLICATION_PROFILE - valid only if th e application_rule is specified as "mustSubmittedApplication" or "mustApprovedApplication"
 
version 
Base Entity  enrollment_restriction 
ColumnName  version 
Definition  
The number of times the record has been updated
