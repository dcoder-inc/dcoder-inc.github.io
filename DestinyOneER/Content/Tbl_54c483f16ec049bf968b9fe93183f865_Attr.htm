instantiation_code 
Base Entity  biz_process 
ColumnName  instantiation_code 
Definition  
Code used to specify the instantiation semantics of this business process. "always_new" means concurrent processes can occur.  "one_at_a_time" means only one instance of this process can exist at one time, multiple processes are executed sequentially.  "only_once" means the process can occur once and only once.
